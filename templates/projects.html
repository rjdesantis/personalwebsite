quickfood

One of my hobbies to watch cooking videos and cook new recipes, so I decided to make quickfood to help! Quickfood is a website that uses spoonacular's api to access over 360,000 recipes. Creating quickfood helped me learn a lot about React, JavaScript, HTML, and CSS while creating something that could be useful for anyone that loves food! Check this out on my linkedin!
